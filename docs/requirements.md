# Customer Manager 要件定義

## 必須機能
- 顧客一覧
- 顧客登録／編集／削除
- ステータス管理（NEW / IN_PROGRESS / CONTRACTED / LOST）
- ランク管理（A/B/C）
- メール・電話・メモの管理

## 非必須（余裕あれば）
- 詳細画面
- updatedAt / createdAt
- 担当者（Staff）テーブルとの紐付け
- 顧客検索（名前 / ステータス）

## 技術スタック
- Spring Boot 3.x
- Thymeleaf
- H2 Database
- Spring Data JPA
- Java 17
